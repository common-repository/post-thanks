== Description ==

Plugin adds custom text message in end of each post published. 

== Installation ==

1. Upload the entire `post-thanks` folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

Do I need to do any setting after installation.

* Go to Wordpress menu 'Post Thank You' after activating plugin. 
* Enter custom text and enter color (Hex codes accepted). 
* Submit to see text in end of all posts.
